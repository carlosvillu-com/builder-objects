# builder-objects

### Instalación:

```bash
$ git clone https://github.com/carlosvillu-com/builder-objects
$ cd builder-objects
$ npm install
```

### Uso

Ejecutar el código de ejemplo para ver como funciona la función asíncrona

```bash
$ npm start
```

Y para ejecutar los tests de ejemplos:

```bash
$ npm run test
```
